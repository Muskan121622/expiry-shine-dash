## Local Setup

### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Quick Start
```
git clone https://github.com/Muskan121622/expiry-shine-dash.git
cd expiry-shine-dash
npm install
npm run dev
```

## Development Workflow

1. **Fork & Clone**
   ```
   git clone https://github.com/YOUR_USERNAME/expiry-shine-dash.git
   git remote add upstream https://github.com/Muskan121622/expiry-shine-dash.git
   ```

2. **Branching**
   ```
   git checkout -b feature/add-login-page
   ```

3. **Commit Messages**
   ```
   feat: add login component
   fix: resolve navbar bug
   docs: update README
   ```

4. **PR Process**
   - Use our [PR template](.github/PULL_REQUEST_TEMPLATE.md)
   - Link issues: `Closes #5`
   - Request review from maintainers

## Using Templates & Labels

- **Issues**: Use `.github/ISSUE_TEMPLATE/` when available
- **Labels**: Check `good first issue`, `help wanted`
- **PRs**: Follow our PR template checklist

## Code Style

- Follow existing patterns
- Run `npm run lint` before commit
- 2-space indentation
- Single quotes for JS
